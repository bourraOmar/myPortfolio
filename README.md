# 🚀 Bourra Omar - Personal Portfolio

Welcome to my personal developer portfolio! This repository contains the source code for my portfolio website, designed to showcase my projects, skills, and professional experience as a Full Stack Web Developer and UI/UX Designer based in Safi, Morocco.

🔗 **Live Demo:** Hosted on GitHub Pages

## 🌟 Features

- **Modern Dark/Gradient UI:** Sleek, modern aesthetic using CSS variables, Flexbox, and CSS Grid.
- **Responsive Design:** Fully responsive layout that adapts gracefully from large desktop displays to mobile phones.
- **Interactive Mockups:** Custom HTML/CSS-built device mockups (MacBook and Smartphones) that act as interactive image sliders to showcase project screenshots cleanly.
- **Performance Optimized:** Includes lazy-loading, asynchronous decoding, and preloading for fast initial load times.
- **Downloadable CV:** Direct PDF download integration.

## 💼 Featured Projects

The portfolio currently highlights several of my key projects:
- **FarmOps Dashboard:** An intelligent real-time farm management solution (NextJS, NestJS, MongoDB).
- **FarmOps Mobile:** A pocket-sized farm management mobile app (React Native, Expo).
- **Auto Rent Pro:** A complete car rental and fleet management platform (Laravel 11, Tailwind CSS, PostgreSQL).

## 🛠️ Built With

- **HTML5:** Semantic architecture and structure.
- **CSS3:** Custom styling, CSS Grid, Flexbox, and fluid animations.
- **Vanilla JavaScript:** Handling interactive image sliders and tab navigation.
- **Font Awesome & SVG:** Vector iconography.
- **Google Fonts:** "Inter" typography.

## 🚀 How to Run Locally

Because this is a beautifully simple static site, no build steps, compiling, or package installations are required!

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Navigate to the project folder:
   ```bash
   cd your-repo-name
   ```
3. Just double-click and open `index.html` in your favorite web browser, or use an extension like Live Server in VS Code.
